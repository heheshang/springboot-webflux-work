* 删除 github 上 .idea文件
```bash
$ git --help # 帮助命令


$ git pull origin master  # 将远程仓库里面的项目拉下来

$ dir # 查看有哪些文件夹

$ git rm -r --cached .idea  # 删除.idea文件夹
$ git commit -m '删除.idea' # 提交,添加操作说明

```
